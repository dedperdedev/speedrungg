# Four Types of Fun — Hard / Easy / People / Serious

Lazzaro's "4 Keys 2 Fun" research (2004). Useful complement to MDA aesthetics.

## The four

| Type | What player wants | Examples |
|---|---|---|
| **Hard fun** | Challenge, mastery, problem-solving | Souls games, Tetris, chess puzzles |
| **Easy fun** | Curiosity, exploration, role-play | Open-world games, narrative adventures |
| **People fun** | Connection, competition, communication | Multiplayer, party games, MMOs |
| **Serious fun** | Meaning, learning, change | Educational games, art games, fitness apps |

## Mapping check (use during creative direction)

- Which of the 4 types does this game serve **primarily**?
- Which one **secondarily**?
- Which two are out of scope?
- Are pillars consistent with the chosen 1–2?

If the answer is "all four equally" — you have no focus, redesign.

## Per-type design implications

### Hard fun
- Difficulty matters most
- Feedback on failure needs to teach
- Mastery curve is the product
- Cosmetics secondary

### Easy fun
- World feel matters most
- Discovery moments need protection (no spoilers in marketing)
- Pacing > challenge
- Optional content > mandatory grind

### People fun
- Communication tools matter (voice, ping, emote)
- Matchmaking quality > graphics
- Anti-toxicity is a first-class feature
- Solo experience usually shallow

### Serious fun
- Theme integrity matters most
- Mechanics serve message
- Audience is narrower but deeper
- Marketing is mission-driven

## Anti-patterns

- Adding a multiplayer mode to a single-player Hard-fun game "for engagement" — usually dilutes
- Educational content jammed into a competitive game — neither serves
- "Cozy multiplayer" without thinking through People-fun-specific design
- Treating Serious fun as a flag instead of as primary aesthetic
