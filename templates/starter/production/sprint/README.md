# production/sprint/

Sprint planning artifacts owned by project-manager.

Files created during /sprint-plan:
- `current.md` — active sprint plan (scope, dates, owner per item)
- `retrospective-YYYY-MM-DD.md` — at sprint end (created via /sprint-retro)
- `velocity.md` — running velocity per type of work

This directory should never be empty after sprint 1.
